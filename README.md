# cloud-computing-
# Multi-Cloud Architecture Using AWS and Azure

## Objective

To demonstrate interoperability between two cloud providers: AWS and Microsoft Azure.

## Architecture

User → AWS Service → Azure Service

## Workflow

1. User sends a request to AWS.
2. AWS forwards the request to Azure.
3. Azure processes the request.
4. Azure returns a response.
5. AWS displays the response to the user.

## Technologies Used

- Python
- Flask
- AWS
- Microsoft Azure
- REST API
